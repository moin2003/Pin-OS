Privacy Policy

Last updated: April 2026  ·  Pin OS Automation

Overview

This application does not collect, store, or share personal user data. Pin OS Automation is designed exclusively for personal automation and testing purposes using Pinterest APIs. Your privacy is a core priority of this tool.

Data Collection

Pin OS Automation does not collect any personally identifiable information (PII). We do not use analytics services, tracking pixels, cookies, or any form of data collection mechanism that captures information about users or their activity.

Authentication & Security

All authentication is handled securely via official OAuth 2.0 methods provided by Pinterest. No credentials, tokens, or sensitive information are stored on external servers. Authentication flows comply fully with Pinterest's developer guidelines and API terms of service.

Data Storage

No sensitive information is stored on external servers. Any operational data required for the automation workflow (such as scheduled pin details) remains local to the user's environment and is not transmitted to or retained by any third-party service.

Third-Party Services

This application interacts solely with Pinterest's official API endpoints. No third-party advertising, analytics, or data-broker services are integrated into this tool. No user data is sold, shared, or disclosed to any external parties.

Scope of Use

Pin OS Automation is intended for personal use. It is not a commercial product offered to the general public, and it is developed and operated in accordance with Pinterest's developer policies and API terms of service.

Contact

If you have any questions regarding this privacy policy or the operation of this application, you may reach out via the contact information provided in the Pinterest developer app listing associated with this tool.
